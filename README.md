# 🛡️ Sentinela Blue Team

## Sobre o projeto

O **Sentinela Blue Team** é uma ferramenta desenvolvida em Python com foco em estudos de **Segurança da Informação**, voltada para práticas defensivas (Blue Team).

O objetivo do projeto é coletar informações básicas do sistema, gerar relatórios de análise e evoluir para uma ferramenta de monitoramento e apoio à segurança.

---

## 🎯 Objetivos

- Praticar programação em Python aplicada à segurança;
- Desenvolver conceitos de defesa cibernética;
- Criar uma ferramenta de análise e monitoramento;
- Aprender boas práticas de desenvolvimento seguro.

---

## 🚀 Funcionalidades atuais

✅ Coleta de informações do sistema operacional;  
✅ Identificação de versão do sistema;  
✅ Identificação do processador;  
✅ Geração de relatório de segurança;  
✅ Organização do projeto utilizando Git.
